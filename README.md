# Task 2: Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data using summary statistics and visualizations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- Titanic-Dataset.csv

## Steps Performed
- Loaded the Titanic dataset.
- Displayed the first five rows of the dataset.
- Checked dataset information and data types.
- Generated summary statistics.
- Identified missing values.
- Created a Histogram for Age distribution.
- Created a Boxplot to detect outliers.
- Created a Correlation Heatmap.
- Created a Pairplot to visualize relationships between features.

## Results
- Missing values were found in the Age, Cabin, and Embarked columns.
- The Age distribution and outliers were visualized.
- Correlation between numerical features was analyzed using a heatmap.
- Pairplot helped understand relationships among important variables.

## Conclusion
The Exploratory Data Analysis was completed successfully using Python libraries. The dataset was analyzed, important patterns were identified, and visualizations were created to better understand the data.
